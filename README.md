# Aufgabenstellung


In diesem Projekt machen wir eine Mehrwertsteuer-Rechner-App.

In Deutschland gibt es zwei Mehrwertsteuersätze: 19 % und 7 %.

Mit dieser App können wir Netto zu Brutto oder Brutto zu Netto rechnen.
